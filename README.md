# spring-cloud-learing
此工程为Spring Cloud 体系研究学习博客配套案例工程资源。Spring Cloud体系中不同组件模块按照文件夹分类。

prometheus：使用对 Prometheus+Grafana 对Spring Cloud项目进行监控

spring boot admin：使用Spring Boot Admin 管理监控Spring Cloud项目

consul模块 ：注册中心扩展（一）consul作为服务注册和发现、配置中心，整合Spring Cloud

gateway：Zuul的升级网关，使用Spring Cloud Gateway作为Spring Cloud项目网关实践

nacos：注册中心扩展（二）nacos作为服务注册和发现、配置中心，整合Spring Cloud

zookeeper：注册中心扩展（三）zookeeper作为服务注册和发现、配置中心，整合Spring Cloud
